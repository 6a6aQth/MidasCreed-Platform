- Opted for Prisma v5 to avoid v7's `prisma.config.ts` complexity without a real database up and running yet.
- Left the actual DB migration (`npx prisma migrate dev` or `db push`) to the user, as it requires a valid Neon PostgreSQL connection string which is currently empty.
